Opdracht 3

Waarom staat de tekst in de blokken horizontaal gecentreerd?
Omdat text align: center; de horizontale uitlijning van tekst binnen een blok element aanpast naar center.
Ga na dat een property vertical-align:center geen effect heeft. Waarom eigenlijk niet?
vertical-align wordt gebruikt voor de verticale uitlijning van inline-elementen en dus niet voor blokelementen.
Wat doet vertical-align dan wel?
vertical-align regelt de verticale positie van inline-elementen ten opzichte van de omliggende tekst of elementen, niet het verticaal centreren binnen een container.
Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren? (tip:line-height)
door line-height gelijk te maken aan de hoogte van het omliggende element.
Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren? (tip: table cell)
Door de vertical-align: middle; eigenschap toe te passen op de tbody van een tabel, kun je de inhoud van meerdere rijen verticaal centreren.
In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang?
De regel voor class 'tall' krijgt voorrang omdat deze specifieker is dan de algemene regel voor class 'area'. CSS geeft voorrang aan meer specifieke regels bij conflicten.
